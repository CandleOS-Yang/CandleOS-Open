# 项目名称
Candle OS(中文：烛光操作系统)

# 项目简介
Candle OS是一个32位图形化操作系统

# 项目目标
- [ \/ ] Boot
    1. 读硬盘
    2. 加载loader
- [ \/ ] Loader
    1. 读硬盘
    2. 读文件
    3. 设置VBE图形模式
    4. 保存VBE信息
    5. 加载并开启保护模式
    6. 读取kernel
- [  ] Kernel
    1. 内存管理初始化 \
    2. GDT初始化     \
    3. IDT初始化     \
    4. PIC初始化     \
    5. 基础输出函数   \
    6. 调试函数       \
    7. 键盘驱动
    8. IDE驱动
    9. FAT16文件系统
    10. 定时器驱动
    11. 任务管理
    12. 图形函数
    13. 图层管理
    14. 窗口管理
    15. 系统循环和消息
    16. ELF解析
    17. 程序加载
    18. API
